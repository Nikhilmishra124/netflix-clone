# 🎬 Netflix Clone

A responsive **Netflix landing page clone** built using **HTML5** and **CSS3**. This project replicates the look and feel of Netflix's homepage while focusing on modern UI design and responsive layouts.

## 🚀 Features

- 🎥 Netflix-inspired landing page
- 📱 Fully responsive design
- 🎨 Modern and clean user interface
- ⚡ Smooth hover effects and transitions
- 📂 Well-structured HTML and CSS code
- 🌐 Cross-browser compatibility

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
